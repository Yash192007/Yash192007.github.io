Embedding a Google Form

1. In Google Forms, open your form, click Send → Embed (<>), copy the `src` URL.
2. Replace the `src` value in the `<iframe>` inside `index.html` (section `#contact`).

Adding your GitHub/LinkedIn

- Update the GitHub links in `index.html` (`your-username`) to your actual GitHub username.
- Update the LinkedIn URL in `index.html` to your profile.
